# Abilities

> **What is this folder?**
> This directory contains raw game data extracts for all spells, abilities, and attacks used by cats, enemies, bosses, and items in the base game.

## File Index

- [`Ability_Pools.md`](./Ability_Pools.md) - Extracted from `data/ability_pools.gon`
- [`Cat_Abilities.md`](./Cat_Abilities.md) - Extracted from `data/abilities/butcher_abilities.gon`, `fighter`, `hunter`, etc.
- [`Enemy_and_Boss_Abilities.md`](./Enemy_and_Boss_Abilities.md) - Extracted from `data/abilities/finalboss`, `kaiju`, `guillotina`, etc.
- [`System_and_Item_Abilities.md`](./System_and_Item_Abilities.md) - Extracted from `data/abilities/armor`, `consumable_item`, `basic_attacks`, etc.
